The Modal service accepts the following parameter as an argument on create(args):

  • config - An object that can contain the following values:
    • Mask - defaults: true - whether to cover the screen with a mask
    • CloseOnMaskClick - defaults: true - whether to close the modal when the mask is clicked on
    • CloseButton - defaults: true - whether to display a close button
    • TemplateUrl - defaults: undefined - URL pointing to a html file that will be rendered inside this modal
    • Draggable - defaults: true - whether the modal should be draggable
    • Escapable - defaults: true - whether the Escape key should close the modal
  • model - This represents the model that will be passed in to render the TemplateUrl specified above
