The slide in card accepts the following parameters as an argument on slideOutCard.push(args);

  • config:
    • timer - the length of time the card displays. Default: 5000
    • templateUrl - URL pointing to the html file that will be rendered inside the slide out card. Default: undefined
    • isTimed - whether the card should use the timer. Default: true
  • model - This represents the model that will be passed in to render the TemplateUrl specified above.
